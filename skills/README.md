# Function Health MCP — Skills

## fh-lab-results (user-invocable)

Interactive skill for querying and analyzing lab results. Invoke with `/fh-lab-results` in Claude Code.

Supports arguments: `summary`, `check`, `sync`, `biomarker <name>`, `changes`, `out-of-range`, `category <name>`, `recommendations`, `report`.

## fh-usage (reference)

Background context skill (non-user-invocable). Automatically loaded to provide context about the data model, auth flow, tool parameters, and common usage patterns.
